# trackrip
_(tracker-rip)_

Extracts samples from music tracker module files.

- Currently supports the following formats:
    - __MOD__
    - __S3M__ (partially)
    - __IT__ (working on it!)

## Usage

`python3 trackrip.py <module_file>`
